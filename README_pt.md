# Gerador de Senhas 2024.

![Status do Fluxo de Trabalho do GitHub](https://github.com/PierreGode/PasswordGenerator/actions/workflows/exefile.yml/badge.svg) ![Status do Fluxo de Trabalho do GitHub](https://github.com/PierreGode/PasswordGenerator/actions/workflows/bandit.yml/badge.svg) [![Última versão do GitHub](https://img.shields.io/github/v/release/PierreGode/PasswordGenerator)](https://github.com/PierreGode/PasswordGenerator/releases) [![Todos os downloads do GitHub](https://img.shields.io/github/downloads/PierreGode/PasswordGenerator/total)](https://github.com/PierreGode/PasswordGenerator/releases)

Esta compilação gera um artefato e 2 arquivos exe.
um aplicativo Windows mais leve com geração de senha aleatória, e outro com tempo de carregamento de 20 segundos devido ao carregamento de transformadores LLM.
<p>
O aplicativo mais pesado tem a adição de geração de frases LLM + "scrambleee", o que significa que tudo é processado no seu próprio computador.
